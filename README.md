# Rule-Mechanism-AI Three-layer Decoupled Architecture

Replication code for:

> "Design and Verification of a Rule-Mechanism-AI Three-layer Collaborative
> Software Architecture for CZ Crystal Growth Process FDC"

---

## Repository Structure

| File | Corresponds to | Description |
|---|---|---|
| `simulation_ablation.py` | Section 6.2, Table 5, Fig. 8–11 | Experiment System A: ablation study (Strategy A–D) |
| `simulation_baseline.py` | Section 6.3, Table 6, Fig. 12–14 | Experiment System B: baseline comparison (Kalman / ARIMA / LSTM) |

## Reproduction

### Simulation experiments (no external data required)

```bash
# Section 6.2 — Ablation study (Table 5, Fig. 8–11)
python simulation_ablation.py

# Section 6.3 — Baseline comparison (Table 6, Fig. 12–14)
python simulation_baseline.py

# Section 6.1 — Latency benchmark (Table 3)
python benchmark_latency.py
```

Simulation data are generated procedurally by the scripts.
No external dataset is needed for the above three scripts.

### Real industrial data experiments

```bash
# Section 6.4 — Rule layer SPC validation (Fig. 15–17)
python rule_layer_spc_validation.py

# Section 6.5 — Cross-furnace aging figures (Fig. 18–19)
python cross_furnace_aging_fig18.py
python cross_furnace_aging_fig19.py
```

**Note:** `rule_layer_spc_validation.py` requires the Device CZ-01 FDC dataset,
and `cross_furnace_aging_fig18.py` / `cross_furnace_aging_fig19.py` require
the B001–B008 cross-furnace FDC dataset (`data.xlsx`).
**These datasets are proprietary industrial data and cannot be made publicly
available due to a confidentiality agreement with the data provider.**
Researchers who wish to reproduce these results may contact the corresponding
author to discuss potential data access arrangements.
The analysis scripts themselves are fully open and can be inspected independently.

## Random Seeds

All simulation experiments use fixed seeds `{0, 1, 7, 42, 123}` for
reproducibility. Minor numerical differences may occur across Python or
library versions.

## Key Parameters

| Parameter | Value | Description |
|---|---|---|
| `G0` | 12.0 K/mm | Simulation: initial thermal gradient baseline |
| `G0` (real data) | 23.553 K/mm | Calibrated from B001 new furnace (body growth phase mean) |
| `VG_CRIT` | 0.065 (sim) / 0.075 (real) | Voronkov critical v/G threshold |
| `Q_KF` | 0.02 | Kalman process noise (mechanism layer dynamic calibration) |
| `R_KF` | 1.0 | Kalman observation noise |
| `SEEDS` | {0, 1, 7, 42, 123} | Fixed random seeds for reproducibility |
| `WARMUP` | 50 samples | Warm-up window before statistics are computed |
